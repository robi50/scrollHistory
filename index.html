    scrollHistory jQuery Plugin: Record Scroll Position!

    Project maintained by robi50 Hosted on GitHub Pages — Theme by mattgraham

    How to use?

    • Include the jQuery Library
    • Include the jQuery Cookie Plugin plugin
    • Include the scrollHistory Plugin
    • Then add this code:
      • $.scrollHistory();
    • You can set the cookie timeout
      • $.scrollHistory(60);
      • "60" means 60 second. If you don't set any value; scrollHistory set default value which mean 999999999999999999
